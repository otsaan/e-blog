## Eblog
School blogging platform.

## Pre-requisits
- Git
- Composer

## Installation
- `git@github.com:otsaan/e-blog.git`
- `cd e-blog`
- `cp .env.example .env`
- `composer install`
- `php artisan migrate --seed`
- `php artisan serve` => [http://localhost:8000](http://localhost:8000)
